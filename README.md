# Shourov Hasan

**Founder & Senior Engineer, NextMind Lab**

Machine Learning Engineer focused on applied AI systems, model development, and scalable ML infrastructure. I have been building in the tech space since 2021 — across research, product, and engineering — with a consistent focus on making intelligent systems that actually work in production.

---

## What I Work On

At NextMind Lab, I lead the engineering side of our ML-first products. My day-to-day spans model training and fine-tuning, designing data pipelines, and shipping AI-powered applications end to end. I care a lot about the gap between research and real-world deployment, and most of what I build lives somewhere in that space.

Outside of the lab, I spend time on language model experimentation, computer vision systems, and low-resource AI — particularly for Bengali and South Asian language contexts.

---

## Core Skills

**Machine Learning & AI**
- Model training, fine-tuning, and evaluation (supervised, self-supervised, RLHF-adjacent workflows)
- Large Language Models — fine-tuning with QLoRA/LoRA, prompt engineering, synthetic data generation
- Computer Vision — object detection, face recognition, real-time inference pipelines
- ML experiment tracking, versioning, and reproducibility

**ML Engineering & Infrastructure**
- Building and maintaining training pipelines from data ingestion to model serving
- Vector databases and retrieval-augmented systems (ChromaDB, FAISS)
- Model deployment and API wrapping (FastAPI, Streamlit)
- Working with GPU-constrained and low-resource environments

**Development**
- Python (primary), JavaScript/TypeScript
- React, Next.js — for building interfaces around AI products
- PostgreSQL, MongoDB, Supabase, Redis
- REST APIs, WebSockets, system design for AI-backed products

**Tools & Platforms**
- PyTorch, TensorFlow, scikit-learn, HuggingFace Transformers
- MLflow, Weights & Biases
- Ollama, LM Studio — local model inference
- Docker, Linux, Git
- AWS, Google Cloud, Cloudflare — for infrastructure and deployment
